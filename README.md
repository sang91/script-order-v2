# Script Order V2

Google Apps Script quản lý đơn Etsy (Key Fob / Strap Watch) + Chrome Extension export.

## Thành phần

| Thư mục / file | Mô tả |
|----------------|--------|
| `*.gs` | Mã nguồn Google Apps Script (14 file) |
| `Export Order Etsy/` | Chrome extension đẩy đơn lên Web App |
| `All Model.txt` | Tra cứu danh sách hãng xe / model / trim |
| [HUONG_DAN_GOOGLE_APPS_SCRIPT.md](./HUONG_DAN_GOOGLE_APPS_SCRIPT.md) | **Hướng dẫn tạo project trên Apps Script & deploy** |

## Bắt đầu nhanh

1. Đọc [HUONG_DAN_GOOGLE_APPS_SCRIPT.md](./HUONG_DAN_GOOGLE_APPS_SCRIPT.md)
2. Copy các file `.gs` lên [script.google.com](https://script.google.com)
3. Deploy Web App → lấy Deployment ID
4. Load extension `Export Order Etsy` → dán Script ID
